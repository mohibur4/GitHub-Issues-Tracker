
    1️⃣ What is the difference between var, let, and const?
        Ans: let – block-scoped, not redeclarable in same scope, no hoisting issues
             const – block-scoped, cannot be reassigned, must be initialized 

    2️⃣ What is the spread operator (...)?
        Ans: expands an array or object into individual elements or properties

     3️⃣ What is the difference between map(), filter(), and forEach()?
        Ans: map – returns a new array after transforming each element
        filter – returns a new array with elements that pass a test
        forEach – executes a function on each element, does not return a new array

    4️⃣ What is an arrow function?
        Ans: concise function syntax using => does not have its own this

    5️⃣ What are template literals?
        Ans: string literals using backticks allowing interpolation ${expression}
